Sistema Geral ADJACARÉ

Portal interno desenvolvido para auxiliar na organização e comunicação entre os departamentos da igreja ADJACARÉ.

O sistema centraliza pedidos de mídia, escalas da equipe e outras informações importantes em um único ambiente, facilitando o fluxo de trabalho entre os ministérios.

Funcionalidades

- Criação de pedidos de mídia
- Kanban para acompanhamento dos pedidos (Pendente / Em produção / Concluído)
- Comentários em cada pedido
- Envio automático de email para a equipe de mídia
- Integração com WhatsApp via Kommo
- Criação e gerenciamento da escala da equipe de mídia
- Painel interno com login de usuários
- Atualizações em tempo real usando Supabase

Tecnologias utilizadas

- React
- Vite
- Supabase
- Vercel
- Kommo CRM
- Resend (envio de emails)

 Estrutura do sistema
- Fluxo: WhatsApp → Kommo → API → Supabase → Vercel → React → Sistema interno

Objetivo
- O objetivo do sistema é melhorar a organização interna da igreja, garantindo que pedidos, escalas e comunicações sejam registrados e acompanhados de forma clara e eficiente.
